# test-my-typing
A light weight cli tool for testing type speed.

# About the project
Built this project to understand how to handle real-time keyboard input and terminal
manipulation using ncurses.
# Tech stack
- Languages: C++
- Library: ncurses
- Build System: cmake

# Getting started
- Prerequisities: 
    - gcc, clangd
    - OS(LINUX)
    - ncurses
    - cmake

1. Clone the repo: 
```
git@github.com:Appyouz/test-my-typing.git
```
 2. Create a directory `build` and change to it
 ```cd build/```

3. Run `cmake --build .`
4. Execute the `./main` program
